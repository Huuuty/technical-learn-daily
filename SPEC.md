# 售前技术知识学习应用 - SPEC.md

## 1. Project Overview

**项目名称**: 技术日知录
**类型**: 单页Web应用 (纯前端)
**核心功能**: 每天为售前工程师推送5个相互关联的技术知识点，覆盖计算机网络、云计算、部署、大数据、AI五大领域，内容通俗易懂。
**目标用户**: 售前工程师，需要向非技术背景客户解释复杂技术概念

---

## 2. Visual & Rendering Specification

### 页面布局
- **整体风格**: 现代卡片式设计，深色主题，灵感来自终端/代码编辑器美学
- **配色方案**:
  - 背景: #0f0f1a (深紫黑色)
  - 卡片背景: #1a1a2e
  - 强调色: #00d4ff (科技蓝)
  - 辅助强调: #ff6b6b (温暖红), #4ecdc4 (薄荷绿), #ffe66d (金黄)
  - 文字: #e8e8e8 (主文字), #888 (次要文字)

### 字体
- 标题: "Noto Sans SC", sans-serif (支持中文)
- 代码/技术术语: "JetBrains Mono", monospace

### 页面结构
1. **顶部导航栏**: 应用标题 + 日期显示 + 刷新按钮
2. **知识卡片区域**: 5张卡片横向/纵向排列，每张代表一个领域
3. **底部**: 领域标签说明 + 版权信息

### 动效
- 卡片入场: 依次淡入上浮，交错延迟100ms
- 悬停: 卡片轻微上浮 + 发光边框
- 刷新: 卡片翻转切换动画

---

## 3. Knowledge Content Specification

### 五大领域
1. **计算机网络** - 颜色标记: #00d4ff (科技蓝)
2. **云计算** - 颜色标记: #4ecdc4 (薄荷绿)
3. **部署** - 颜色标记: #ffe66d (金黄)
4. **大数据** - 颜色标记: #ff6b6b (温暖红)
5. **AI** - 颜色标记: #a855f7 (紫色)

### 知识点数量
- **计算机网络**: 15个
- **云计算**: 12个
- **部署**: 12个
- **大数据**: 12个
- **AI**: 15个
- **总计**: 66个知识点

### 每日推送逻辑
- 每次展示10个知识点
- 确保每个领域至少1个
- 自动建立知识点之间的"联系链"说明
- **去重机制**: 使用 localStorage 记录当天已展示的知识点
  - 每天自动重置（根据日期判断）
  - 展示过的知识点不会重复出现
  - 连续点击"换一批"3次可强制重置

---

## 4. Interaction Specification

### 用户交互
- **刷新按钮**: 随机获取新的5个知识点组合
- **卡片点击**: 展开详细解释（模态框）
- **标签筛选**: 点击顶部领域标签可高亮该领域卡片

### 响应式设计
- 桌面端: 5卡片横向排列
- 平板端: 2-3列网格
- 手机端: 单列纵向滚动

---

## 5. Technical Implementation

### 技术栈
- 纯HTML + CSS + JavaScript
- 无需后端，数据全部内置在前端
- 使用CSS Grid/Flexbox布局
- CSS动画实现过渡效果

### 数据结构
```javascript
{
  id: number,
  domain: "network" | "cloud" | "deployment" | "bigdata" | "ai",
  title: string,           // 知识点标题
  analogy: string,         // 生活比喻（让非技术人员理解）
  core: string,             // 核心技术点
  connection: string,      // 与其他知识点的联系
  keywords: string[]       // 关联关键词
}
```

---

## 6. Acceptance Criteria

1. 页面加载后显示5个知识点卡片
2. 每个卡片显示领域、标题、比喻说明
3. 点击卡片展开详细解释
4. 刷新按钮可随机切换知识点组合
5. 所有界面文字为中文
6. 动效流畅，无明显卡顿
7. 知识点之间有明确的联系说明
8. 适配桌面端和移动端
